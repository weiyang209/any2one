调研深度

选择你的调研深度

让研究模式匹配任务需求，在覆盖度、成本与报告深度之间取得平衡。

模式 · simple
快速概览
快速了解背景并补充核心相关工作，适合轻量汇报或前期探索。
3–5 papers ≥3 opened 2 recent
模式 · medium
均衡研究
兼顾覆盖度、成本和报告深度，适合作为多数任务的默认研究模式。
6–10 papers ≥6 opened 4 recent
模式 · complex
深度研究

10–15 papers ≥10 opened 6 recent

输入支持

支持的输入格式

音频、视频、文档、表格、演示、压缩包与在线链接均可自动识别，并路由到合适的处理流程。

🎙️
会议音频
→ 结构化会议笔记
.mp3 .wav .m4a
使用 WhisperX 进行语音识别和说话人分离，自动生成带说话人归属的 7 段式结构化笔记。
🎬
会议视频
→ 结构化会议笔记
.mp4 .mov .mkv
使用 ffmpeg 提取音频流，再接入 meeting-audio-grounding 流程，生成完整结构化会议笔记。
📄
文档材料
→ 结构化研究笔记
.pdf .docx .md .txt
Docling 深度提取文本、表格、图像、公式与代码块，生成 8 段式结构化笔记。
📊
表格数据
→ 数据分析笔记
.xlsx .csv
提取字段结构、统计摘要、预览行，并自动生成可视化，识别异常值与数据质量问题。
📑
演示文稿
→ 演示结构笔记
.pptx
提取页面结构、标题、正文、图表、表格与备注，分析叙事脉络和关键信号。
📦
压缩归档
→ 档案级汇总笔记
.zip
解压并识别各子文件类型，并行调用相应 grounding skill，最后生成跨文件汇总笔记。

快速开始

开始上手

三步，从原始输入到完整研究报告。

one-report · Prompt 示例 
# 将下面这段 prompt 发给 Cursor Agent

请使用 .cursor/skills/one-report/
来生成完整报告。

输入：
  input_path： docs/showcase/inputs/case5/HER2-case5.zip
  output_formats： pdf
  research_mode： complex

搜索设置：
  search_backend：  cursor
  require_open_link： true
  download_opened_literature： true

可选项：
  output_lang： zh
  transcription_language： zh
1

准备输入文件

将文件放入 data/raw_inputs/。支持音频、视频、文档、表格、PPT/PPTX、压缩包，也支持直接粘贴链接。

2

把 prompt 发给 Cursor Agent

阅读 .cursor/skills/one-report/SKILL.md，按示例填写 input_path、research_mode 和 output_formats。

3

等待全流程执行

Agent 会自动完成 grounding → research → summary → review → export 的端到端执行。

输出路径
data/grounded_notes/<id>/grounded.md
data/lit_results/<id>/lit.md
data/report_inputs/<id>/summary.md
data/reports/<id>/research_report.md
data/final_outputs/<id>/report.*
